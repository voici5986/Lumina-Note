# Lumina Note 项目

## 项目简介

Lumina Note 是一款 AI 驱动的本地笔记应用，专注于知识管理和创作辅助。

## 技术栈

| 层级 | 技术 |
|------|------|
| 前端 | React 18 + TypeScript |
| 后端 | Rust + Tauri 2.0 |
| 数据库 | SQLite (向量存储) |
| AI | 多模型支持 (OpenAI/Claude/本地) |
