# Agent 评估系统

## 目标

构建一套完整的 Agent 能力评估体系，用于：
1. 测试 Agent 任务完成能力
2. 发现系统问题
3. 指导优化方向

## 评估维度

### 1. 任务完成度 (权重 35%)
- 最终状态检查
- 里程碑完成率
- 结果质量

### 2. 工具正确性 (权重 25%)
- 工具选择准确率
- 参数准确率
- 执行成功率

### 3. 计划质量 (权重 25%)
- 步骤合理性
- 计划遵循度
- 效率性

### 4. 效率 (权重 15%)
- Token 消耗
- 响应时间
- 循环次数
